# Lesson 1: Logic of Data (Guided Demonstration Resources)

This folder supports **Lesson 1: The Logic of Data Analysis**.

In Lesson 1, the notebook and sample files are used for a **guided overview demonstration** of the IPO workflow:
- **Input**: identify the data source and what the table represents,
- **Process**: discuss how analysts inspect and organize data,
- **Output**: view an example visualization and interpret what it communicates.

## Lesson Intent for This Folder
This folder is not the full technical CSV-analysis lesson. It exists to provide a simple, stable demo resource while students focus on foundational concepts and vocabulary.

Deeper hands-on notebook mechanics are taught in Lesson 2, and deeper CSV-centered analysis is taught in Lesson 3.

## Files
- `notebook.ipynb`: Guided demonstration notebook used in Lesson 1.
- `data_en.csv`: Supporting English CSV used for the demo context.
- `data_fr.csv`: Supporting French CSV used for the demo context.

## Source Acknowledgement
